一个基于 Python 开发的经典扫雷游戏，具有现代化的图形界面和丰富的游戏功能。

## ✨ 特性

-  **经典玩法** - 忠实还原 Windows 经典扫雷游戏
-  **美观界面** - 使用 Tkinter开发的图形化界面
-  **多难度选择** - 初级、中级、高级三种游戏难度
-  **变种扫雷训练** -支持找漏雷，直线判雷，定式判雷三种训练模式
